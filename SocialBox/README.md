# Installation
```
apt-get update
apt-get install git
git clone https://github.com/GHOSTs-OCTOPUS/Hack-With-Android/SocialBox.git
cd SocialBox-Termux
chmod +x install-sb.sh
./install-sb.sh
```
# Run
```
*bash SocialBox.sh
*./SocialBox.sh
```
